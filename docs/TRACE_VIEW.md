# Trace View

## Purpose
Provide operational transparency, debugging support, and user trust.

## Trace items
- user input received
- interpreted intent
- selected action
- model call started
- model call completed
- graph update
- storyboard job queued
- storyboard job completed
- Story Sentinel warning
- export artifacts created

## Non-goals
- does not claim to reveal hidden model reasoning
- does not visualize latent space
- does not simulate mechanistic interpretability
