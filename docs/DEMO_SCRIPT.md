# Demo Script

## Goal
Show one strong, coherent loop.

## Sequence
1. Open StoryForge
2. Speak a story idea
3. Director responds
4. Trace View shows intent + actions
5. Story graph updates
6. Storyboard frames generate
7. User interrupts and refines
8. Story Sentinel flags a continuity issue
9. Enter Character Interview Mode
10. Ask the character a question
11. Export output

## What judges should notice
- live multimodal interaction
- interruption support
- coherent canvas updates
- trace transparency
- clear authoring value
